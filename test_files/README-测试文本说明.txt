智学助手 - 测试文本使用说明
============================

本文件夹包含不同学科的测试文本，可用于测试智学助手的问答功能。

文件列表：
---------
**学科分类：**
1. 数学-三角函数.txt      - 包含三角函数基础公式和特殊角数值
2. 英语-语法从句.txt      - 包含名词性从句、定语从句、状语从句的语法知识
3. 政治-马克思主义.txt    - 包含马克思主义哲学、政治经济学基本原理
4. 计算机-Python基础.txt  - 包含Python编程语言基础知识

**其他分类：**
5. 其它-学习方法与时间管理.txt
6. 其它-职业规划与技能发展.txt
7. 其它-心理健康与情绪管理.txt
8. 其它-创新思维与问题解决.txt
9. 其它-科技前沿与未来趋势.txt

使用方法：
--------
1. 打开智学助手小程序
2. 进入首页，点击"上传学习资料"
3. 选择聊天文件，上传上述文本文件
4. 上传完成后，进入"对话"页面进行问答测试

建议测试问题：
------------
数学：
- 什么是正弦函数？
- sin30度等于多少？
- sin²x + cos²x = ?

英语：
- 什么是定语从句？
- who和which有什么区别？
- 主语从句怎么用？

政治：
- 哲学的基本问题是什么？
- 对立统一规律是什么？
- 社会发展的动力有哪些？

计算机：
- Python中的列表和元组有什么区别？
- 什么是面向对象编程？
- 如何定义一个函数？
