## 基本信息
- **开发平台**：微信小程序

## API接口对接

### 后端接口地址
```javascript
// config/api.js
const API_BASE = 'http://localhost:8000'; // 开发环境
// const API_BASE = 'https://your-server.com'; // 生产环境

module.exports = {
  // 文档相关
  UPLOAD_DOC: `${API_BASE}/api/documents/upload`,
  GET_DOCS: `${API_BASE}/api/documents`,
  DELETE_DOC: `${API_BASE}/api/documents/{id}`,
  
  // 对话相关
  CHAT: `${API_BASE}/api/chat`,
  GET_HISTORY: `${API_BASE}/api/history`,
  
  // 用户相关
  LOGIN: `${API_BASE}/api/auth/login`,
  GET_PROFILE: `${API_BASE}/api/user/profile`
}
```

### 关键API调用示例

**1. 上传文档**
```javascript
uploadDocument(filePath) {
  wx.uploadFile({
    url: API.UPLOAD_DOC,
    filePath: filePath,
    name: 'file',
    header: {
      'Authorization': `Bearer ${wx.getStorageSync('token')}`
    },
    success: (res) => {
      const data = JSON.parse(res.data);
      if (data.success) {
        wx.showToast({ title: '上传成功' });
        this.loadDocuments(); // 刷新列表
      }
    },
    fail: (err) => {
      wx.showToast({ title: '上传失败', icon: 'error' });
    }
  });
}
```

**2. 发送消息并获取AI回答**
```javascript
async sendMessage(question) {
  this.setData({ isLoading: true });
  
  try {
    const res = await wx.request({
      url: API.CHAT,
      method: 'POST',
      header: {
        'Authorization': `Bearer ${wx.getStorageSync('token')}`,
        'Content-Type': 'application/json'
      },
      data: {
        document_id: this.data.currentDoc.id,
        question: question
      }
    });
    
    // 添加AI回答到消息列表
    const aiMessage = {
      type: 'ai',
      content: res.data.answer,
      sources: res.data.sources,  // 参考来源
      time: new Date().toLocaleTimeString()
    };
    
    this.setData({
      messages: [...this.data.messages, aiMessage],
      isLoading: false
    });
    
  } catch (error) {
    wx.showToast({ title: '请求失败', icon: 'error' });
    this.setData({ isLoading: false });
  }
}
```

## 开发规范

### 命名规范
- **页面**：小写，用连字符连接，如 `document-list`
- **组件**：大驼峰，如 `MessageBubble`
- **变量**：小驼峰，如 `userName`
- **常量**：大写下划线，如 `MAX_FILE_SIZE`

### 样式规范
```css
/* 使用 rpx 作为单位 */
.container {
  padding: 32rpx;
  font-size: 28rpx;
}

/* 颜色变量（在app.wxss定义） */
page {
  --primary: #07c160;
  --secondary: #576b95;
  --background: #f5f5f5;
  --text: #333333;
}
```

### 代码规范
```javascript
// 使用 async/await 处理异步
async function fetchData() {
  try {
    const res = await api.getData();
    return res.data;
  } catch (error) {
    console.error('获取数据失败:', error);
    throw error;
  }
}

// 错误处理统一封装
function handleError(error) {
  wx.showToast({
    title: error.message || '操作失败',
    icon: 'error'
  });
}
```

## 常见问题

### Q1: 如何调试网络请求？
**A**: 使用微信开发者工具的Network面板，或在代码中加`console.log`。

### Q2: 文件上传大小限制？
**A**: 微信小程序限制单次上传50MB，建议后端限制20MB。

### Q3: 如何存储用户登录状态？
**A**: 使用`wx.setStorageSync`存储token，请求时从header发送。

### Q4: 页面间如何传递数据？
**A**: 小数据用URL参数，大数据用全局状态管理或`wx.setStorage`。
