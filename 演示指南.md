## ⚡ 一分钟启动

### 选择启动模式

本项目有两种后端模式：

| 模式 | 脚本 | 特点 | 适合场景 |
|------|------|------|---------|
| **RAG 完整模式** | `start_rag.bat` | 加载 RAG 模型，语义检索回答 | **推荐！答辩演示用** |
| **简单模式** | `start_simple.bat` | 不加载 RAG，本地关键词检索 | 仅前端调试 |

**请使用 `start_rag.bat`**（RAG 完整模式）进行答辩演示。

### RAG 完整模式（RAG 模式，推荐）

双击 `start_rag.bat`，等待出现：
```
[4/4] 启动 FastAPI 服务...
============================================================
  🎉 智学助手 RAG 完整模式已启动！
  API地址: http://localhost:8000
  API文档: http://localhost:8000/docs
============================================================
```
即可。不要关闭这个窗口。

**或者手动运行：**
```bash
python start_rag.py
```

**RAG 模式会自动：**
1. 加载本地 Embedding 模型（BAAI/bge-small-zh）
2. 从数据库读取已上传文档并重建向量索引
3. 启用 RAG 问答（未配置 DeepSeek API Key 时自动降级为本地语义检索）

**手动运行（RAG 完整模式）：**
```bash
python start_rag.py
```

### 第二步：打开微信开发者工具

1. 打开微信开发者工具
2. 点击「导入项目」
3. 项目目录选择：`d:\Code\Projects\zhixue-assistant\frontend`
4. AppID 填写已有的测试号或点击「测试号」
5. 点击确定，等待编译完成

> ⚠️ **重要设置**：右上角「详情」→「本地设置」→ 勾选「**不校验合法域名**」

## 🎯 答辩演示流程（建议顺序）

### 1. 展示首页状态（30秒）
- 打开小程序，首页显示「后端状态：本地检索演示模式」
- 说明：系统已连接后端，支持文件上传和智能问答

### 2. 上传学习资料（1分钟）
- 点击首页「上传学习资料」
- 选择「从聊天记录选择」→ 上传 `test_files/machine_learning.txt`
- 展示弹窗：**文件名、分类：计算机、置信度：XX%**
- **重点说明**：这里用的是**朴素贝叶斯分类器**，自动识别文档学科类型

### 3. 展示资料库（30秒）
- 切换到「资料库」Tab
- 展示文件卡片，显示分类标签（紫色渐变徽章）
- 点击某个文件，展示操作菜单（查看详情/预览/删除）

### 4. 智能问答演示（2分钟）
- 切换到「对话」Tab
- 输入问题：**"什么是监督学习？"**
- 展示回答：显示回答内容 + 📌 本地知识库检索标签 + 参考来源

- 再问：**"深度学习和神经网络有什么关系？"**
- 展示来源追溯功能

### 5. 展示对话历史（30秒）
- 进入首页，点击「对话历史」快捷入口
- 展示历史记录列表（问题摘要 + 时间 + 继续对话按钮）

### 6. 展示「我的」页面（30秒）
- 切换到「我的」Tab
- 点击「AI系统信息」，展示：
  - RAG系统：fallback_mode（本地模式）
  - 文档分类器：ready（朴素贝叶斯）
  - 技术细节：Embedding / FAISS / DeepSeek
- 说明：系统设计了完整的 RAG 架构，本次演示为本地检索模式

## 🤖 AI 方法说明（答辩重点）

| AI方法 | 所在模块 | 作用 | 输入/输出 |
|--------|----------|------|-----------|
| **朴素贝叶斯分类器** | `ai_module/document_classifier.py` | 自动对上传文档分类 | 文本 → 类别+置信度 |
| **关键词检索（本地RAG降级）** | `backend/main.py` | 基于文档内容回答问题 | 问题+文档库 → 答案+来源 |
| **RAG（检索增强生成）架构** | `ai_module/` | 完整版：向量化检索+大模型生成 | 需配置DeepSeek API Key |

## 🔧 常见问题

**Q: 小程序报"网络请求失败"**
→ 确认后端在运行（8000端口），检查「不校验合法域名」已勾选

**Q: 上传文件后分类显示"其他"**
→ 正常，上传纯数字/空白文件会分类到"其他"；有实际内容的文档会正确分类

**Q: 问答回答说"没有匹配内容"**
→ 先上传相关文档，再提问；关键词要出现在文档中

**Q: 后端启动报错**
```bash
pip install fastapi uvicorn python-multipart scikit-learn PyPDF2
```
