## 产品简介

**智学助手**是一款基于RAG（检索增强生成）技术的智能学习助手微信小程序。它能够帮助学生高效管理学习资料，并通过AI技术实现基于资料的智能问答。

### 什么是RAG技术？
RAG（Retrieval-Augmented Generation，检索增强生成）是一种结合信息检索和文本生成的AI技术。简单来说：
- **先检索**：从你的学习资料中找到相关内容
- **再生成**：基于找到的内容，由AI生成精准回答
- **可溯源**：每个回答都标明来源，确保准确性

### 适用场景
- 📚 复习备考：快速查找知识点，答疑解惑
- 📝 论文写作：整理参考文献，提取关键信息
- 💡 知识管理：分类存储学习资料，构建个人知识库
- 🤔 深度学习：通过问答加深对知识的理解

## 快速开始

### 第一步：启动后端服务

在电脑上打开命令行，进入项目目录：

```bash
cd D:\Code\Projects\zhixue-assistant
python start_simple.py
```

看到以下信息表示启动成功：
```
Server starting...
API URL: http://localhost:8000
Docs URL: http://localhost:8000/docs
```

### 第二步：启动小程序

1. 打开微信开发者工具
2. 导入 `frontend` 文件夹
3. 点击"编译"按钮
4. 在模拟器中看到小程序界面

### 第三步：开始使用

1. **上传资料**：点击首页"上传学习资料"按钮
2. **开始提问**：切换到"对话"页面，输入问题
3. **查看历史**：在"对话历史"页面查看过往问答

## 功能详解

### 1. 首页

首页是小程序的入口，提供以下功能：

#### 上传学习资料
- 点击大按钮"上传学习资料"
- 支持两种方式：
  - **从聊天记录选择**：选择微信聊天中的文件
  - **从相册选择**：选择图片资料

#### 快捷入口
- **我的资料库**：查看所有上传的文件
- **开始提问**：直接进入对话页面
- **对话历史**：查看过往问答记录

#### 最近上传
- 显示最近上传的5个文件
- 显示文件名、分类、上传时间
- 下拉刷新获取最新列表

#### 功能介绍
- 文档上传说明
- AI问答说明
- 知识溯源说明
- 智能分类说明

### 2. 资料库

资料库是管理所有学习资料的地方。

#### 文件列表
- 以卡片形式展示所有文件
- 显示文件图标、分类标签、文件名、大小、时间
- 支持搜索功能

#### 文件操作
点击文件卡片，可选择：
- **查看详情**：显示文件完整信息
- **预览文件**：查看文件内容（支持PDF/TXT）
- **删除文件**：从系统中删除该文件

#### 搜索功能
- 在顶部搜索框输入关键词
- 实时过滤文件列表
- 支持按文件名搜索

#### 上传资料
- 点击顶部"上传资料"按钮
- 快速跳转到首页上传功能

### 3. 对话（智能问答）

对话页面是核心功能，基于你的学习资料进行智能问答。

#### 开始对话
1. 在底部输入框输入问题
2. 点击"发送"按钮或按回车
3. 等待AI生成回答

#### 问答界面
- **用户消息**：右侧蓝色气泡，显示你的问题
- **AI消息**：左侧白色气泡，显示回答内容
- **参考来源**：AI回答下方显示引用的文档
- **加载状态**：AI思考时显示动画

#### 快捷操作
- **长按复制**：长按消息可复制内容
- **清空对话**：点击右上角垃圾桶图标清空当前对话
- **自动滚动**：新消息自动滚动到底部

#### 使用建议
- 问题越具体，回答越准确
- 上传相关资料后再提问效果更佳
- 可以追问，继续深入讨论

### 4. 对话历史

保存所有问答记录，方便回顾。

#### 历史列表
- 显示所有问答记录
- 显示问题、回答摘要、时间
- 按时间倒序排列（最新的在前）

#### 操作功能
- **查看详情**：点击记录查看完整问答
- **继续对话**：跳转到对话页面，基于该问题继续提问
- **清空历史**：一键清空所有历史记录
- **下拉刷新**：更新历史列表

### 5. 我的（个人中心）

个人中心提供系统信息和设置功能。

#### AI系统信息
- 查看RAG系统状态
- 查看文档分类器状态
- 了解技术详情

#### 功能按钮
- **清空对话历史**：清除所有问答记录
- **清除缓存**：清除小程序本地缓存
- **关于我们**：查看版本信息和开发者

## 使用技巧

### 如何获得更好的AI回答？

1. **上传相关资料**
   - 确保上传与问题相关的文档
   - 文档内容越丰富，回答越准确

2. **提问要具体**
   - ❌ 不好："讲讲机器学习"
   - ✅ 好："什么是监督学习？请举例说明"

3. **分步提问**
   - 复杂问题拆分成多个简单问题
   - 逐步深入，层层递进

4. **利用历史记录**
   - 在历史记录中继续对话
   - 避免重复提问相同问题

### 文件管理建议

1. **合理命名文件**
   - 使用有意义的文件名
   - 如："高等数学_第一章_极限.txt"

2. **定期整理资料库**
   - 删除不需要的文件
   - 保持资料库整洁

3. **分类上传**
   - 按科目整理文件
   - 便于后续查找

### 提高搜索效率

1. **使用关键词**
   - 输入文件名的关键词
   - 支持模糊匹配

2. **结合分类**
   - 先查看分类标签
   - 再使用搜索功能

## 常见问题

### Q1: 上传文件后没有反应？

**可能原因**：
- 后端服务未启动
- 网络连接问题
- 文件格式不支持

**解决方法**：
1. 检查命令行是否显示 `Server starting...`
2. 确认使用的是支持的格式（PDF、TXT、Word）
3. 查看微信开发者工具控制台是否有错误信息

### Q2: AI回答不准确？

**可能原因**：
- 未上传相关资料
- 资料内容不包含答案
- 问题过于宽泛

**解决方法**：
1. 先上传与问题相关的学习资料
2. 检查资料中是否包含相关内容
3. 将问题问得更具体一些

### Q3: 小程序提示"网络错误"？

**可能原因**：
- 后端服务停止
- 防火墙阻止连接
- IP地址配置错误

**解决方法**：
1. 重新启动后端服务
2. 暂时关闭Windows防火墙
3. 检查 `app.js` 中的 `apiBaseUrl` 是否为 `http://localhost:8000`
4. 在微信开发者工具中勾选"不校验合法域名"

### Q4: 文件预览失败？

**可能原因**：
- 文件格式不支持预览
- 文件已损坏
- 网络问题

**解决方法**：
1. 目前仅支持TXT和PDF预览
2. 确保文件完整未损坏
3. 重新上传文件

### Q5: 如何删除已上传的文件？

**方法**：
1. 进入"资料库"页面
2. 点击要删除的文件
3. 在弹出菜单中选择"删除文件"
4. 确认删除

### Q6: 支持哪些文件格式？

**目前支持**：
- ✅ TXT（纯文本）
- ✅ PDF（便携文档格式）
- ✅ Word（DOC/DOCX）
- ⚠️ 图片（仅支持上传，不建议用于问答）

**建议**：
- 纯文本文件效果最佳
- PDF需要可提取文本（非扫描版）

### Q7: 文档分类不准确怎么办？

**原因**：
- 文档内容较短
- 内容涉及多个学科
- 专业术语较少

**解决方法**：
- 这是正常现象，不影响问答功能
- 可以在资料库中查看实际分类

### Q8: 如何备份我的数据？

**数据位置**：
- 数据库文件：`zhixue.db`
- 上传的文件：`uploads/` 文件夹

**备份方法**：
1. 复制 `zhixue.db` 文件
2. 复制 `uploads/` 整个文件夹
3. 保存到安全位置

## 技术架构

### 前端
- **平台**：微信小程序
- **技术**：WXML + WXSS + JavaScript

### 后端
- **框架**：FastAPI（Python）
- **数据库**：SQLite
- **AI引擎**：RAG系统

### AI技术栈
| 组件 | 技术 | 说明 |
|------|------|------|
| 文本向量化 | BGE-small-zh | 将文本转为向量 |
| 向量检索 | FAISS | 高效相似度搜索 |
| 重排序 | BGE-reranker | 精排检索结果 |
| 文本生成 | DeepSeek API | 生成自然语言回答 |
| 文档分类 | 朴素贝叶斯 | 自动分类文档 |
