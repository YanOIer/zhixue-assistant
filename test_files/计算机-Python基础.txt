Python 编程基础
================

一、变量和数据类型

1. 基本数据类型：
- 整数 (int)：如 1, 100, -5
- 浮点数 (float)：如 3.14, -0.5
- 字符串 (str)：如 "Hello", 'Python'
- 布尔值 (bool)：True, False

2. 类型转换：
int() - 转换为整数
float() - 转换为浮点数
str() - 转换为字符串

二、运算符

1. 算术运算符：
+ 加法
- 减法
* 乘法
/ 除法
// 整除
% 取余
** 幂运算

2. 比较运算符：
== 等于
!= 不等于
> 大于
< 小于
>= 大于等于
<= 小于等于

3. 逻辑运算符：
and - 逻辑与
or - 逻辑或
not - 逻辑非

三、流程控制

1. 条件语句：
if 条件1:
    代码块1
elif 条件2:
    代码块2
else:
    代码块3

2. 循环语句：
for 变量 in 序列:
    代码块

while 条件:
    代码块

3. 循环控制：
break - 跳出循环
continue - 跳过本次循环

四、数据结构

1. 列表 (list)：
fruits = ["apple", "banana", "orange"]
列表操作：
- append() 添加元素
- insert() 插入元素
- remove() 删除元素
- sort() 排序

2. 元组 (tuple)：
coordinates = (10, 20)
特点：不可变

3. 字典 (dict)：
person = {"name": "Tom", "age": 20}
字典操作：
- keys() 获取所有键
- values() 获取所有值
- items() 获取所有键值对

4. 集合 (set)：
nums = {1, 2, 3, 4, 5}
特点：无序、不重复

五、函数

定义函数：
def 函数名(参数):
    函数体
    return 返回值

参数类型：
- 位置参数
- 关键字参数
- 默认参数
- 不定长参数 *args, **kwargs

六、面向对象

类定义：
class 类名:
    def __init__(self, 参数):
        self.属性 = 参数
    
    def 方法(self):
        方法体

三大特性：
- 封装
- 继承
- 多态

七、异常处理

try:
    可能出错的代码
except 异常类型:
    处理异常
finally:
    无论是否异常都会执行

八、文件操作

读取文件：
with open("file.txt", "r") as f:
    content = f.read()

写入文件：
with open("file.txt", "w") as f:
    f.write("Hello World")
