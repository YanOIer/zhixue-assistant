高等数学学习笔记

第一章：极限与连续

极限是微积分的基础概念。函数f(x)在x趋近于a时的极限L，记作：
lim(x→a) f(x) = L

重要极限公式：
1. lim(x→0) sin(x)/x = 1
2. lim(x→∞) (1 + 1/x)^x = e
3. lim(x→0) (e^x - 1)/x = 1

第二章：导数

导数描述函数的变化率。函数f(x)在点x处的导数定义为：
f'(x) = lim(h→0) [f(x+h) - f(x)] / h

求导法则：
1. 常数法则：(c)' = 0
2. 幂函数法则：(x^n)' = nx^(n-1)
3. 和差法则：(u ± v)' = u' ± v'
4. 乘积法则：(uv)' = u'v + uv'
5. 商法则：(u/v)' = (u'v - uv')/v²

第三章：积分

积分是导数的逆运算，分为不定积分和定积分。

基本积分公式：
1. ∫x^n dx = x^(n+1)/(n+1) + C (n≠-1)
2. ∫1/x dx = ln|x| + C
3. ∫e^x dx = e^x + C
4. ∫sin(x) dx = -cos(x) + C
5. ∫cos(x) dx = sin(x) + C

定积分的几何意义是曲边梯形的面积。

第四章：微分方程

微分方程是含有未知函数及其导数的方程。

常见类型：
1. 可分离变量方程
2. 一阶线性微分方程
3. 二阶常系数线性微分方程

应用领域：物理学、工程学、经济学等。
